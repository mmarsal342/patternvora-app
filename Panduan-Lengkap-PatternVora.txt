üìù BAGIAN 4: TEXT OVERLAY
================================================================================

Fitur text overlay punya 3 mode:

MODE 1: NORMAL (Text on Top)
- Text ditampilkan di atas pattern dengan warna solid
- Bisa atur warna, opacity, blend mode
- Cocok untuk watermark, branding

MODE 2: CLIP (Pattern Inside Text)
- Pattern di-crop/mask ke dalam text
- Text jadi "window" ke pattern
- Background tampil, pattern hanya di area text
- Cocok untuk typographic art

MODE 3: MOSAIC (Shapes Form Text) - NEW!
- Shapes kecil disusun membentuk text
- Efek pointillism/mosaic art
- Atur Mosaic Density untuk kepadatan shapes
- Tidak ada regular pattern, hanya mosaic shapes
- Cocok untuk artistic typography

CARA PAKAI:
1. Expand "Text Overlay" panel
2. Centang "Enable Text (Current Layer)"
3. Ketik text yang diinginkan
4. Pilih font dari dropdown
5. Pilih "Pattern Fill Mode": Normal/Clip/Mosaic
6. Atur Size dan Opacity
7. Jika Mosaic: atur Mosaic Density (0.5x - 2.5x)

üé¨ BAGIAN 5: ANIMASI & VIDEO EXPORT
================================================================================

PatternVora bisa export video animasi loop!

JENIS ANIMASI PRIMARY:
- None: Tidak ada animasi
- Orbit: Shapes berputar mengelilingi center
- Float: Shapes melayang naik-turun
- Scan: Efek scanning left-to-right

JENIS ANIMASI SECONDARY:
- None: Tidak ada
- Pulse: Shapes membesar-mengecil
- Spin: Individual shapes rotate

ANIMATION SETTINGS:
- Duration: Panjang video (detik)
- Speed: Kecepatan animasi
- Intensity: Intensitas movement

CARA BUAT VIDEO:
1. Expand "Motion" panel
2. Toggle ON "Animation"
3. Pilih Primary Animation (Orbit/Float/Scan)
4. Optional: Pilih Secondary Animation (Pulse/Spin)
5. Atur Duration, Speed, Intensity
6. Klik tombol "MP4" di footer
7. Konfirmasi recording tips
8. Tunggu processing (progress bar akan tampil)

TIPS VIDEO:
- Duration 3-5 detik cukup untuk seamless loop
- Jangan minimize browser saat recording
- Chrome recommended untuk best quality
- Video auto seamless loop!

üì± BAGIAN 11: TIPS & BEST PRACTICES
================================================================================

UNTUK SOCIAL MEDIA:
- Instagram Post: Aspect Ratio 1:1
- Instagram Story: Aspect Ratio 9:16
- Facebook: Aspect Ratio 1.91:1
- YouTube Thumbnail: Aspect Ratio 16:9
- Export HD atau 4K untuk sharp quality

UNTUK MICROSTOCK (Adobe Stock, Shutterstock):
- Export 4K (4096px) untuk max resolution
- Enable "Stock-Ready" checkbox pada SVG
- Avoid seasonal/trademark shapes
- Buat variasi dengan Shuffle & Batch Studio
- 10+ variasi dari 1 design = portfolio cepat

UNTUK PRINT:
- Export SVG untuk vector (infinite scale)
- Atau export 4K PNG minimum
- Pattern otomatis seamless/tileable!
- CMYK conversion di Illustrator/Photoshop

PERFORMANCE TIPS:
- Complexity tinggi = rendering lambat
- Truchet/Guilloche intensive on GPU
- Kurangi complexity saat editing preview
- Naikkan complexity sebelum final export
- Close other tabs saat video recording

‚ùì TROUBLESHOOTING
================================================================================

Pattern tidak muncul / blank canvas?
- Refresh halaman (Cmd+R / Ctrl+R)
- Clear browser cache
- Pastikan JavaScript enabled
- Try different browser (Chrome recommended)

Export gagal / stuck?
- Kurangi complexity
- Close other browser tabs
- Pastikan browser up-to-date
- Try smaller export size first

Video export sangat lama?
- Normal untuk high complexity + long duration
- Jangan tutup atau minimize tab
- Progress bar menunjukkan status
- Gunakan "Fast" quality mode untuk preview
- Reduce duration atau complexity

Text tidak tampil?
- Pastikan font ter-load (tunggu 2-3 detik)
- Try font "Inter" atau "Outfit" (always available)
- Check opacity slider tidak 0

Shapes tidak bisa di-drag?
- Pastikan layer tidak locked
- Enable "Edit Shapes" mode di footer
- Klik shape dulu untuk select
